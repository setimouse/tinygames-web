## 1. 产品概述

TinyGames 经典游戏合集官网，展示九款经典游戏（数独、扫雷、2048、俄罗斯方块、打砖块、五子棋、国际象棋、中国象棋、西洋跳棋）汇聚一盒的移动应用。以霓虹玻璃拟态视觉风格，传达无广告无内购、回归纯粹游戏乐趣的品牌理念。

- **目标用户**: 喜欢益智和棋类游戏的玩家，追求纯净游戏体验的用户
- **核心价值**: 游戏种类丰富、AI引擎离线对战、无广告无内购

## 2. 核心功能

### 2.1 页面规划

1. **首页**：品牌形象展示、核心卖点、游戏概览网格、下载引导
2. **游戏介绍页**：九款游戏的详细介绍、玩法说明、截图展示
3. **关于/下载页**：应用详情、功能特色列表、下载链接

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero区域 | 应用名称、标语、霓虹玻璃拟态背景动画 |
| 首页 | 游戏网格 | 9款游戏的图标网格展示，悬停有发光效果 |
| 首页 | 特色亮点 | 无广告/离线对战/AI引擎三大特色卡片 |
| 首页 | 下载引导 | CTA按钮引导下载 |
| 首页 | 页脚 | 版权信息、社交链接 |
| 游戏介绍页 | 游戏详情卡 | 每款游戏的玩法说明、截图、特色标签 |
| 游戏介绍页 | 分类导航 | 单人游戏/双人对弈分类切换 |
| 关于页 | 应用简介 | 版本信息、开发理念 |
| 关于页 | 特色列表 | 详细功能特色展示 |
| 关于页 | 下载区 | 各平台下载链接 |

## 4. 用户界面设计

### 4.1 设计风格

- **设计主题**: 霓虹玻璃拟态 (Neon Glassmorphism)
- **主色调**: 深色基底 (#0a0a1a)，霓虹蓝 (#00d4ff)，霓虹紫 (#7c3aed)，霓虹粉 (#ff3ea5)
- **辅助色**: 毛玻璃半透明白色 (rgba(255,255,255,0.05~0.15))
- **字体**: 
  - 标题: "Orbitron" (科技感无衬线字体)
  - 正文: "Inter" (清晰易读)
- **按钮样式**: 霓虹发光边框 + 毛玻璃填充，悬停时亮度增强
- **布局风格**: 全屏沉浸式布局，卡片式内容展示
- **动效**: 滚动渐入、悬停发光、霓虹闪烁、毛玻璃视差

### 4.2 页面设计

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | Hero | 全屏深色背景、霓虹发光标题、动态粒子/网格背景、CTA按钮 |
| 首页 | 游戏网格 | 3×3玻璃态卡片网格，每张卡片含游戏图标和名称 |
| 首页 | 特色亮点 | 水平排列的三张特色卡片，带霓虹边框 |
| 游戏介绍页 | 分类Tab | 单人游戏/棋类游戏切换标签 |
| 游戏介绍页 | 游戏卡片 | 大尺寸详情卡片，含游戏截图、玩法描述、AI标签 |
| 关于页 | 内容区 | 应用描述、版本号、技术栈、下载链接列表 |

### 4.3 响应式设计

- 桌面优先设计，桌面端展示完整布局
- 平板端调整卡片间距和网格列数
- 移动端单列布局，优化触控

## 5. 品牌调性

- **品牌色**: 深色背景搭配霓虹发光色彩，强调科技感与纯粹感
- **语气**: 专业、简洁、专注游戏体验本身
- **差异化**: 霓虹玻璃拟态设计语言 + 无广告纯净体验的强调
